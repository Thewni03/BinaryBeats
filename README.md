**Binary Beats** is a music mobile application developed with **Kotlin ** in ** Android Studio**.

The app allows users to explore music, browse song albums, personalize their playlists based on beats, and even interact with a built-in chatbot for a smarter music experience.

It is designed to make music listening more interactive, customizable, and engaging.

--------

## Features

* Music Home Page
* Song & Album Page
* Personalized music selection based on beats
* Custom music management
* Built-in Chatbot for user interaction
* Clean and modern UI design

## Tech Stack

* **Language:** Kotlin
* **IDE:** Android Studio
* **UI Design:** XML Layouts
* **Platform:** Android


## How to Run the Project

Make sure you have installed:

* Android Studio (Latest version recommended)
* Android SDK
* Git

#### step 1 - Clone the Repository

```bash
git clone https://github.com/Thewni03/BinaryBeats.git
```

#### step 2 - Run the App

* Connect your Android device OR start an Emulator
* Click the **Run** button
* Select your device
* The app will install and launch automatically
